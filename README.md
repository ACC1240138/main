## Work in Progress

**Microsimulation modeling of alcohol policy in Chile**

This repository contains the core analytical work of the FONDECYT project. It includes R scripts for data preparation, mortality trend analysis, and estimation of population-attributable fractions (PAF) and potential impact fractions (PIF) for alcohol-related harms in Chile. The work focuses on developing a microsimulation model to evaluate the effect of policy changes on alcohol use and related harm across different population subgroups.

Status: *In Development*
